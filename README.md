# tabemasu : a dong train exec sample project
## Installation
Please use [pipenv](http://docs.pipenv.org/en/latest/)
Under the project folder
```sh
$ pipenv install
```

## Usage
### Training
Under the project folder
```sh
$ pipenv run tabemasu
```
### Demo Prediction
Under the project folder
```sh
$ pipenv run tabemasu --test_predict
```
